PROJECT GAME PLAN – MIST SINGLE-CELL FINAL

⸻

	1.	FINAL QUESTION

⸻

How do infection conditions affect cell type composition and marker gene expression, and how confidently can these cell types be distinguished based on their gene signatures?

⸻

	2.	OVERALL GOAL

⸻

Use standard single-cell RNA-seq analysis (Seurat workflow) to:
	1.	Identify cell types
	2.	Compare how they change across infection conditions
	3.	Evaluate how clearly (confidently) these cell types can be identified based on marker genes

⸻

	3.	DATASET SETUP

⸻

	•	Load Seurat object (MIST dataset)
	•	Check metadata:
	•	infection variable (e.g., infection_status)
	•	tissue (optional if you want to subset)
	•	clusters (seurat_clusters)

Commands to inspect:
colnames(seurat_obj@meta.data)
table(seurat_obj$infection_status)

⸻

	4.	ANALYSIS STEPS

⸻

STEP 1 – Baseline clustering visualization
	•	Plot UMAP colored by cluster
Goal: Understand overall cell populations

STEP 2 – Identify marker genes
	•	Use FindAllMarkers()
	•	Select top 3–5 marker genes per cluster
Goal: Define what each cluster represents biologically

STEP 3 – Assign rough cell types
	•	Use known marker genes (e.g. CD3D, EPCAM, LYZ)
	•	Label clusters if possible
Goal: Interpret clusters biologically

STEP 4 – Compare across infection conditions
	•	Plot UMAP split by infection
Goal: See how cell populations shift visually

STEP 5 – Cell type composition analysis
	•	Calculate proportion of each cluster per infection group
	•	Create stacked bar plot
Goal: Quantify how infection changes cell populations

STEP 6 – Marker gene expression comparison
	•	Create DotPlot or FeaturePlot for key genes
Goal: Show how gene expression differs across clusters

STEP 7 – Confidence / interpretability analysis (NOVEL PART)
	•	For each cluster:
	•	Evaluate strength of marker expression
	•	Look for:
	•	strong, specific markers → high confidence
	•	weak or overlapping markers → low confidence

Optional simple metric:
	•	average expression of top markers per cluster

Goal: Identify which cell types are clearly defined vs ambiguous

	6.	KEY MESSAGE (RESULTS)

Infection alters both:
	1.	Cell type composition

	•	Others become less distinct, suggesting ambiguity in cell identity
